# CaptureVoca
영어 지문에 형관펜에 줄을 친후 사진을 찍으면, 형광펜 표시가 된 단어들만 골라 단어장을 만들어 주는 서비스

# 주요 기능
- 단어 뜻 자동완성
- 형광펜 표시 기반 단어장 생성
- 단어장을 PDF 파일로 내보내기

# 기술 스택
- Google MLKit
- Jetpack Compose
